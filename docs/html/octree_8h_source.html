17  * BANTER FOLLOWS!
18  *
19  * So, Octrees are cool. Our one needs to be dynamic and loose. This means that:
20  *
21  * 1. If an object is added outside the root node, we need to adjust for it. The steps in this case are:
22  *
23  * while new_object.outside(root):
24  * new_root = OctreeNode(width=root.width*2, position=root.position - direction_of_object)
25  * new_root.add_child(root)
26  * root = new_root
27  * Basically, we dynamically grow upwards towards the object.
28  *
29  * 2. Each node of the octree should be loose. Which means that the bounds of the node are half the size
30  * of the parent, rather than a quarter of the size.
31  */

141 /* A dynamic, loose Octree implementation. Things to note:
142  *
143  * * The tree can grow upwards and downwards as needed. Calling grow(Boundable*) will determine
144  * if nodes need to be added above or below the current tree
145  * * In the event that an object exists in the cross-over between nodes, the one containing the
146  * centre point within it's "strict" bounds will take it.
147  * * When an object changes location and moves outside its node, the tree will be recursed upwards
148  * until a parent is found that contains the object, and then recurse downwards to find the new
149  * node. If the object moves outside the root node this will require the tree to grow upwards. It
150  * is slower to move an object a great distance than a small one.
151  * * Objects will be stored as far down the tree as they will fit
152  * * If an object is added outside the bounds of the root node, the tree will grow in the direction
153  * of the new object until the root node encompasses it. This means that the Octree can move around in
154  * space. For example a fleet of spaceships moving a great distance will cause the root node to grow
155  * large, and then as child nodes empty shift in the direction of the fleet.
156  * * If a node has no objects, and no children, it is removed. If the root node has no objects and only
157  * one child, then the child becomes the new root.
158  */
